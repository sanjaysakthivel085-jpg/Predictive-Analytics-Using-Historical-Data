# Predictive Analytics Using Historical Data

## Project Overview
This project predicts future sales using historical sales data and Linear Regression.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Features
- Data Cleaning
- Data Visualization
- Sales Forecasting
- Future Trend Prediction
- Model Evaluation

## Dataset
sales_data.csv

The model successfully learned the sales trend from historical data and predicted future sales values.
